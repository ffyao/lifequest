## 截图规范

1. **命名**：严格按上表文件名保存为 `.png` 格式。
2. **分辨率**：建议 1920×1080 或浏览器全屏截图。
3. **内容**：截图需包含完整的浏览器窗口或 AI 对话窗口，避免裁剪关键信息。
4. **顺序**：按序号 01-12 顺序保存，便于答辩时按流程展示。
5. **脱敏**：如有 API Key、密码等敏感信息，截图前需打码处理。
