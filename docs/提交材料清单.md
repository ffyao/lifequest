# LifeQuest 作业提交材料清单

本清单依据《大模型编程-提交物要求&评分标准》整理，用于打包前自查。

## 1. ZIP 压缩包

提交用途：提交文档、截图和代码文件，压缩包大小不超过 100M。

提交入口：

```text
https://www.ssfssp.com/sspApp/internship?id=189
```

命名格式：

```text
组号-答辩汇报人姓名.zip
示例：第X组-唐汉.zip
```

压缩包建议包含：

```text
lifequest/
├── README.md
├── package.json
├── client/
├── server/
├── docs/
│   ├── 规格说明.md
│   ├── 架构设计.md
│   ├── 审查报告.md
│   ├── 操作手册.md
│   ├── 协作文档.md
│   ├── API接口规范.md
│   ├── AI-Prompt设计.md
│   └── 提交材料清单.md
└── screenshots/
    └── README.md
```

不建议打包：

```text
.git/
server/data/lifequest.sqlite
真实 DeepSeek API Key
浏览器缓存、临时日志、大体积录屏
```

说明：如果课程平台只要求提交文档和截图，代码文件为非必交项；本项目仍建议打包源码，方便评审方本地运行。

## 3. PPT / PDF

PPT 需要单独提交并转为 PDF，不放入 ZIP 内，PDF 大小不超过 50M。

提交入口：

```text
https://www.ssfssp.com/sspApp/internship?id=190
```

命名格式：

```text
组号-答辩汇报人姓名.pdf
示例：第X组-唐汉.pdf
```

PPT 建议结构：

1. 项目背景与选题动机。
2. 小组分工与 Gitee 分支协作。
3. 技术方案与架构图。
4. AI 辅助开发流程与关键 Prompt。
5. 核心功能演示截图。
6. 代码审查与测试结果。
7. 遇到的问题与解决方式。
8. 总结与收获。
