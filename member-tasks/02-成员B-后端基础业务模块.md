你是 LifeQuest 项目的后端基础业务 AI。你的目标是补齐基础 CRUD 能力、提高接口稳定性，并补充必要测试。

重点成果：

- 补全目标更新和删除接口。
- 补全任务编辑接口。
- 保持现有核心流程测试通过。
- 不改变 XP、等级、徽章核心规则。

## 2. 通用执行流程

## 4. 任务 B1：补全目标更新接口

新增或完善：

```text
PUT /api/goals/:id
```

请求体支持：

```json
{
  "title": "30 天掌握 Vue 项目开发",
  "description": "完成基础语法、组件通信和项目实战。",
  "category": "学习",
  "status": "active"
}
```

要求：

- 只能更新当前用户自己的目标。
- 标题不能为空。
- 返回格式：

```json
{
  "goal": {}
}
```

## 5. 任务 B2：补全目标删除接口

新增或完善：

```text
DELETE /api/goals/:id
```

要求：

- 只能删除当前用户自己的目标。
- 删除后不要导致任务列表接口报错。
- 可以保留任务并将 `goalId` 设为 `NULL`，因为数据库外键已配置 `ON DELETE SET NULL`。
- 返回格式：

```json
{
  "deleted": true
}
```

## 6. 任务 B3：补全任务编辑接口

新增或完善：

```text
PUT /api/tasks/:id
```

请求体支持：

```json
{
  "title": "更新后的任务标题",
  "description": "更新后的任务说明",
  "difficulty": "normal",
  "dueDate": "2026-07-10"
}
```

要求：

- 只能修改当前用户自己的任务。
- 不允许通过该接口修改 `xpReward`、`status`、`completedAt`。
- 不允许让已完成任务再次获得 XP。
- 返回格式：

```json
{
  "task": {}
}
```
